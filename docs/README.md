## 阿里云OSS支持与文件处理增强

我们激动地向您宣布，本次更新正式引入了对**阿里云OSS（对象存储服务）**的全面支持！这不仅扩展了我们的存储选项，还带来了更灵活的文件管理能力。

> **重要提示**：此次更新需要您在配置文件中添加 `ali-oss` 依赖，并确保正确配置相关参数。

### 主要更新内容：

- 🆕 **新功能**:
  - 新增了 **阿里云OSS** 作为可选的文件存储后端。
  - 更新了配置文件模板，新增了 `ali-oss` 相关的配置项，方便您快速集成。
- ✨ **功能优化**:
  - **重构了文件处理逻辑**，使其能够根据配置自动适配不同的存储类型（本地、OSS等）。
  - 优化了**设置页面**，新增了专门用于**添加和管理OSS配置**的模块，让存储管理变得**直观且简单**。
  - *用户体验* 得到显著提升，现在您可以更轻松地在不同存储方案之间切换或管理。

### 如何开始使用？

1. 在您的配置文件中引入 `ali-oss` 依赖。
2. 根据指引配置您的阿里云OSS访问密钥、bucket名称等必要信息。
3. 在设置页面中完成OSS的详细配置，即可开始使用OSS存储服务。

我们相信，这些改进将为您的应用带来更高的灵活性和更优的性能表现！
